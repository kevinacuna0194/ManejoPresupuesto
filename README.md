# ManejoPresupuesto

Este proyecto va a ser una aplicación de manejo de presupuestos a través de esta aplicación.
Los usuarios van a poder manejar sus finanzas, además de poder autenticar si los usuarios van a poder registrar sus ingresos y gastos, también van a poder registrar sus cuentas, tarjetas, cuentas de banco efectivo y demás para tener un registro preciso de sus actividades económicas.
Además, van a poder registrar categorías para que puedan saber en qué gastan su dinero, comida, renta, etc..
Ofreceremos a nuestros usuarios una herramienta de reportaría donde podrán visualizar sus resúmenes diarios, semanales y mensuales.
También realizaremos un calendario que permitirá al usuario visualizar los ingresos y gastos del mes de un vistazo.
Todo esto requiere el uso de una base de datos en donde los usuarios van a poder registrar toda esta información.
Finalmente, permitiremos a los usuarios exportar su reporte de ingresos y gastos a Excel.
